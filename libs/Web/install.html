Install dependencies of the game


The necessary dependencies for the correct operating of the game are pygame and moviepy, libraries of python, and the driven graphing utility, gnuplot.

For install, follow the instructions in this page or go to the game, in the settings menu you gonna get the installation option.


Automatically Instalation

  1. Open the game, go to settgins menu and enter in Install necessary dependencies
  2. Install python, don't forget to check the box to add python to path.
  3. Install gnuplot, don't forget to check the box to add python to path.

Manual Instalation

Python

Python is an interpreted, high-level, general-purpose programming language.For the game it is necessary for the control and reproduction of the audio and video effects.

  1. Install the latest version of python or for the automatically download of the windows version click here .
  2. On the instalation window don't forget to check the box to add python to path. If you want to add python to path at another moment you can do it with these instructions
  3. Open terminal and execute the next command:
    pip install pygame moviepy

Gnuplot

Gnuplot is a portable command-line driven graphing utility for Linux, OS/2, MS Windows, OSX, VMS, and many other platforms. The source code is copyrighted but freely distributed (i.e., you don't have to pay for it). It was originally created to allow scientists and students to visualize mathematical functions and data interactively, but has grown to support many non-interactive uses such as web scripting. It is also used as a plotting engine by third-party applications like Octave. For this game, it is necessary to plot the lauch graphics.

  1. Install the latest version of gnuplot or for the automatically download of the windows version click here .
  2. On the instalation window don't forget to check the box to add gnuplot to path.
